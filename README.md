QuickFIX/Go [![GoDoc](https://godoc.org/github.com/quickfixgo/quickfix?status.png)](https://godoc.org/github.com/quickfixgo/quickfix)
===========

Open Source [FIX Protocol](http://www.fixprotocol.org/) library implemented in Go

FIX versions 4.0-5.0

Example Apps
------------

See [examples](https://github.com/quickfixgo/examples) for some simple examples of using QuickFIX/Go.

Build
-----

The unit tests depend on [gocheck](http://labix.org/gocheck). Acceptance tests depend on ruby in path.

    make
    make check
